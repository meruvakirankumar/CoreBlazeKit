# Components / Grid

The core `DataGrid<T>` component and its parts.

Planned:
- `DataGrid<T>` — main grid with sorting, filtering, paging, inline & in-cell editing.
- `GridColumn<T>` — declarative column definition (Field, Title, Template, EditTemplate, Sortable, Filterable).
- `GridCommandColumn<T>` — command buttons (Edit / Delete / Save / Cancel / Add).
- `GridPager` — pagination UI.
- Row-state tracking via `GridEditContext<T>` (Added / Edited / Deleted).
